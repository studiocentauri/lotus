class Lotus::Entity¶

Lightweight wrapper class to represent entities.

This class is the main interface through which the user should manipulate entities. It provides methods to manipulate components and destroy the entity. The class wraps an entity handle and a registry pointer, and is therefore lightweight enough to be passed around.
