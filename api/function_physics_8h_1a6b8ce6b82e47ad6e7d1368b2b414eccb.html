Matrix4f Lotus::LPerspective(float fov, float aspectRatio, float zNear, float zFar)¶

Calculate a transformation matrix to bring the object into clip space with a perspective projection.

Return

Transformation matrix

Parameters

  • fov: Half-angle field of view in degrees

  • aspectRatio: Aspect ratio of the projection

  • zNear: Frustum near cut-off

  • zFar: Frustum far cut-off
